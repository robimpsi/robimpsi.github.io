# Inventory Forecasting Dashboard

This dashboard helps store managers predict stock levels for the upcoming quarter.

## Features

- **Automated Data Ingestion**: Pulls sales data from CSV exports.
- **Seasonality Adjustment**: Accounts for holiday spikes.
- **Low Stock Alerts**: Highlights items below safety stock levels.
